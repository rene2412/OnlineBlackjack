# OnlineBlackjack

## Overview
This is a complete Blackjack game engine implemented entirely in C++. It supports:
- Player vs Dealer gameplay
- Split hands
- Betting system
- AI dealer logic
- Modular and maintainable code structure

## Features
- Object-oriented design using classes for Player, Dealer, and Game
- Smart pointers (shared_ptr) for memory management
- Data structures and algorithms for game state management
- Optional WebSocket / HTTP integration for multiplayer (if applicable)
- Playable via command line or optional GUI wrapper
