# OnlineBlackjack

## Overview
This is a complete Blackjack game engine implemented entirely in C++ and React. It supports:
- Player vs Dealer gameplay
- Split hands
- Betting system
- AI dealer logic
- Modular and maintainable code structure

## Features
- Object-oriented design using classes for Player, Dealer, and Game
- Smart pointers (shared_ptr) for memory management
- Data structures and algorithms for game state management
- WebSocket / HTTP integration (multiplayer coming soon)
